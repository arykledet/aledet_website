Hello World

How to write a hello world program in C

Here is the source code required to print hello world on your terminal.

#include "stdlib.h"
#include "stdio.h"

int main(int argc, char* argv[])
{
    printf("Hello World!\n");
    return 0;
}

Now to run the code we must compile it, in this instance we are saving the program as hello_world.c and using the GNU C Compiler (GCC) to build the binaries.

gcc -o hello_world hello_world.c

All you have to do now is run the program!

./hello_world
